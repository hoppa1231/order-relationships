# Отношения порядка

## Описание проекта
- Данная работа реализует отношения порядка
- Возможносто создавать и редактировать отношение
- Производить простые вычисления (+, -, *, /)
- Консольный интерфейс с панелью управления
- Защита от всевозможного некорректного ввода

## Архитектура
- `manual` - руководство пользователя
- `presentation` - презентация программы с описанием задачи
- `release` - содержит программу для запуска
- `src`, `src-interface` - исходные коды проекта

## Требования
- Запускать рекомендуется под Windows, в Linux несколько иначе отображается панель управления
